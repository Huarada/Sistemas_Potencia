Instruções do software:

Esse programa resolve EXCLUSIVAMENTE o caso de carga estrela isolada (com o Nó N' não aterrado)

Todos os valores pedidos deverão ser preenchidos em NOTAÇÃO CARTESIANA (A +jB) !!! Assim, é necessário transformar as tensões de fase dos geradores nessa notação (Real[V] = V*cos(fase); Real[V] = V*sin(fase) )
Não é preciso de () para inserir valores totalmente reais. Ex: para inserir Z = 45 basta escrever 45 e enter

Os dados deverão ser preenchidos dessa forma: Z= (Zreal, Zim), dando "enter" para cada input - MESMA COISA PRAS TENSÕES!!!
Exemplo: Para preencher dados de Z para carga = 90 + j45; j50; j40,7 =>     (90,45)      
                                                                            (0,50)
USEM PONTO PARA SEPARAR DECIMAL, CASO CONTRARIO O PROGRAMA ENCERRARÁ ==>   (0,40.7) 


Após incluidos todos os dados, será mostrado na tela as matrizes com Impedancia de Carga, Impedancia de linha, Modulo das tensões inseridas (CONFIRA se inseriu os dados e fez a notação cartesiana corretamente),juntamente com as respostas das Correntes


NOTA: Para achar as tensões de fase nas cargas em triangulo é preciso TRANSFORMAR AS IMPEDANCIAS DE CARGA EM DELTA-ESTRELA antes de inserir os valores. FAZER E SOLUCIONAR MANUALMENTE a transformação estrela-triangulo para as tensões de fase nas cargas
      esse programa apenas calculará as correntes de linha desse circuito específico.



RELEMBRANDO: AS CORRENTES SÃO ENTREGUES EM NOTAÇÃO CARTESIANA!!!!!!!!!!!!!!!!!!!!!!!!!
